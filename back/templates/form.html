Chrust

[xrust]

Educational system empowering user with ability to explore audio signals through machine learning, data engineering and Rust. The idea was to allow everyone to explore how neural networks work, what kind of data they consume, how data is extracted, view the data and explore it across five different genres.

Origin: All of the models that were developed for this system were trained and assembled using Torch library.
The part of the system where user interacts with the models,
as well as inference of the transformed data, is written using Rust. Adding these two, we get Chrust.
